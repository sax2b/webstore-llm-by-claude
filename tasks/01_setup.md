# Task 01: Scaffolding Vite React+TypeScript Application with TailwindCSS

## 🎯 Objective

Initialize a new Vite application using React and TypeScript from an absolute blank directory, install Tailwind CSS dependencies, and verify the local development pipeline functions.

## 📁 Connected References
- **Automation Guides:** `AGENTS.md`

---

## 🧱 Implementation Requirements & Steps

### 1. Scaffold the Core Application Frame
- Execute the native Vite creation command to scaffold a fresh React + TypeScript application in the current directory (`.`):
  
```bash
  npm create vite@latest . -- --template react-ts
```
(Note: Select 'y' to proceed if prompted by the CLI framework.)

- Install the baseline package dependencies generated by the scaffolding engine:
```bash
  npm install
```

### 2. Install Tailwind CSS & Initialize Configuration Files
- Install the core Tailwind compilation engines and layout tools as development dependencies:
```bash
  npm install -D tailwindcss postcss autoprefixer
```
- Generate the system structural configuration scripts:
```bash
  npx tailwindcss init -p
```

### 3. Establish Flat File Architecture Layout
Wipe standard starter templates and establish the strict folder layouts dictated in AGENTS.md:

- Create the structural directories required by your system rulebook:
```bash
  mkdir -p src/components
  mkdir -p src/api
  touch src/api/api.ts
```
- Open src/App.tsx, wipe its pre-existing template code completely, and replace it with a clean placeholder script layout:
```typescript
export default function App() {
    return (
      <div className="p-8 text-center">
        <h1>Vite + React + TypeScript Scaffolding Complete</h1>
      </div>
    );
  }
```
